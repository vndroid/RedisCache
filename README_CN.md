# RedisCache

一款 Typecho 缓存插件。

## 简介

当前语言： [英文](/README.md) | **简体中文**

### 插件亮点

- 支持 Typecho 1.2+ 版本
- 遵循官方插件开发规范，使用命名空间代替老方法；
- 通过访问路径来对文章内容进行缓存，提高重复访问速度；
- 轻量级设计，性能更优；

### 使用方法

下载压缩包或者克隆项目到插件目录 `usr/plugins/`，插件目录名必须为 `RedisCache`，然后在后台开启插件即可。

### 更新方法

更新插件前注意先禁用插件，再执行更新。

如果使用 `git` 方式安装，在此插件根目录中执行命令更新：

```bash
git pull --rebase
```
